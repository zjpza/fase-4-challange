# src/dashboard
Interface visual da solucao.

- `app.py`: aplicacao principal (Streamlit/Flask).
- `index.html`: layout alternativo se usar HTML puro.

Telas:
- Mapa de risco por equipamento/regiao.
- Alertas em tempo real.
- Evolucao temporal do risco.
- Filtros por persona (operador/gestor).

Para executar:
```bash
streamlit run app.py
```
