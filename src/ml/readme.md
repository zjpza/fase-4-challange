# src/ml
Modelos preditivos e avaliacao estatistica.

- `01_eda.ipynb`: analise exploratoria de dados.
- `02_modelagem.ipynb`: treinamento de modelos ML.
- `03_avaliacao.ipynb`: matriz de confusao, metricas, curva ROC.
- `04_predict.py`: script de predicao para novos dados.
- `models/`: modelos serializados (.pkl).
- `README_ML.md`: resumo de algoritmos e metricas.
